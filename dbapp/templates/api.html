Документация MyPosts API

Используйте эти методы для интеграции со сторонними сервисами или автоматизации.

GET /api/list

Получение списка всех опубликованных постов в формате JSON.

Пример ответа (JSON):

[
  {
    "header": "Название поста",
    "desc": "Текст описания поста..."
  }
]

POST /api/post

Создание новой публикации в базе данных.

Параметры запроса (Form-Data):

Параметр Тип Обязательный Описание
header string Да Заголовок нового поста.
desc string Нет Текст или описание поста.

Пример запроса через cURL:

curl -X POST -H "Content-Type: application/json" -d "{\"header\":\"API работает!\", \"desc\":\"Данный пост выложен через API через cURL\"}" http://127.0.0.1:8000/api/post/
